# IJCAI-18-TIANCHI

## 1、赛题解读
### 根据用户历史行为，构建预测模型预估用户的购买意向，即给定广告点击相关的用户（user）、广告商品（ad）、检索词（query）、上下文内容（context）、商店（shop）等信息的条件下预测广告产生购买行为的概率。

## 3、solution文件夹
